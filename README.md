## Описание

Программа для учёта рабочего времени. Позволяет добавлять записи о задачах, подсчитывать общее количество часов и выводить все записи.

## Функции

- `add_entry(date, task, hours)` – добавляет запись.
- `total_hours()` – возвращает сумму часов.
- `list_entries()` – печатает все записи.

## Инструкция по запуску

1. Скопируйте файл `timesheet.py` в свою папку.
2. В Python используйте:

```python
from timesheet import add_entry, total_hours, list_entries

add_entry("2025-05-29", "Написание отчёта", 4)
add_entry("2025-05-30", "Проверка кода", 2)
print(total_hours())   # 6
list_entries()

```

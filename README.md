Описание

Проект представляет собой простую программу для учёта рабочего времени. Реализованы три основные функции:

    add_entry(date, task, hours) – добавляет запись о выполненной работе.

    total_hours() – возвращает общее количество часов за все записи.

    list_entries() – выводит все записи на экран.

Как запустить

Скопируй файл timesheet.py в свой проект. Затем в Python можно использовать:

from timesheet import add_entry, total_hours, list_entries

add_entry("2025-05-29", "Написание отчёта", 4)
add_entry("2025-05-30", "Проверка кода", 2)
print(total_hours())
list_entries()
